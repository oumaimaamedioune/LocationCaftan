# Application de Gestion de Location de Caftans 

## Description
Cette application Android est conçue pour simplifier la gestion d'une boutique de location de caftans. 
Elle permet à l'administrateur de gérer l'inventaire et de suivre les modèles disponibles de manière intuitive.

## Fonctionnalités Principales
 **Authentification Admin :** Sécurisation de l'accès aux fonctionnalités de gestion.
 **Tableau de Bord :** Interface centrale pour naviguer entre l'ajout et la consultation des articles.
 **Gestion du Catalogue :** Ajout de nouveaux caftans avec leurs détails (titre, prix, image).
 **Liste Dynamique :** Affichage de tous les caftans enregistrés via une `RecyclerView`.

## Structure Technique
 **Langage :** Java (Android SDK).
 **Réseau :** Utilisation de Retrofit pour la communication avec l'API.
* **Affichage d'images :** Intégration de la bibliothèque Glide pour le chargement fluide des photos.
